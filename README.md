## Getting Started

1. Clone the repository:
```bash
git clone [your-repo-url]
```

2. Install dependencies:
```bash
npm install
pip install -r requirements.txt
```

3. Run the development server:
```bash
npm run dev
python app.py
```


4. Build for production (might have to change later):
```bash
npm run build
```
